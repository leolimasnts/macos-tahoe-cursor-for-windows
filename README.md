1. Download or clone this repository
2. Open the `cursors` folder
3. Right-click on `Install.inf`
4. Select **Install**
5. Go to **Settings > Bluetooth & devices > Mouse > Additional mouse settings**
6. Select the **Pointers** tab
7. Choose **MacOS-Tahoe-White** from the Scheme dropdown
8. Click **Apply**
